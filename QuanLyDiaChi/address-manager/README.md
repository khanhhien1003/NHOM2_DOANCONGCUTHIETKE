# Address Manager

## Overview
The Address Manager is a simple web application designed to help users manage their addresses efficiently. Users can add, update, and delete addresses through a user-friendly interface.

## Project Structure
```
address-manager
├── src
│   ├── index.html       # Main HTML document
│   ├── style.css        # CSS styles for the application
│   └── app.js           # JavaScript code for managing addresses
├── README.md            # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd address-manager
   ```
3. Open the `src/index.html` file in your web browser to view the application.

## Usage
- To add a new address, click on the "Add Address" button and fill in the required details.
- To update an existing address, click the "Update" button next to the address you wish to modify.
- To delete an address, click the "Delete" button next to the address you want to remove.
